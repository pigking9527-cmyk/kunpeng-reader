# English Frequency List

`frequent_en_10000.txt` was generated from `wordfreq` 3.1.1 using
`top_n_list("en", ...)`, then filtered to lowercase ASCII alphabetic entries
that exist in this project's embedded ECDICT-derived English dictionary.

- Project: https://github.com/rspeer/wordfreq
- Software license: Apache License 2.0
- Frequency data attribution: see the `wordfreq` project documentation and
  its stated Creative Commons Attribution-ShareAlike 4.0 data terms.

The generated list is used only to choose the order in which optional local
word-pronunciation cache files are created.
