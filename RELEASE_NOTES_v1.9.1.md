# 鲲鹏阅读器 v1.9.1

v1.9.1 是面向 Windows 与 macOS 的维护版本。

## 主要更新

- Windows 10/11：提供 64 位安装包和便携版。
- macOS：首次提供适用于 Apple Silicon（M 系列芯片）的版本。
- 阅读器：菜单、弹层和工具栏状态统一管理，阅读模式切换更稳定。
- 界面：设置窗口默认宽度调整为 600 px。
- 同步：服务器 HTTPS 证书已恢复，并启用自动续期。

## macOS 安装提示

macOS 包当前使用临时签名，尚未经过 Apple 公证。首次启动如被系统拦截，请在“系统设置 → 隐私与安全性”中允许鲲鹏阅读器打开。

## 校验

每个发布资产均附带 SHA-256 校验和。安装前请用 `SHA256SUMS.txt` 核对下载文件。
