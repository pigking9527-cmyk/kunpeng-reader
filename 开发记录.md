# 开发记录 / CHANGELOG

鲲鹏阅读器（Kunpeng Reader）开发期间对功能的增删与调整记录。

---

## 开发中

### ✨ 新增 / 改进

- 语义索引设置页接入新的 `semantic_tasks` 任务中心视图，后端统一输出语义模型、语义索引、加速索引的任务状态、进度、按钮可用性和操作文案；旧 `semantic_status` 保留兼容。
- 新增 `scripts/release.ps1` 一键发布脚本，可串联检查、绿色版构建、NSIS 安装包、ASCII 资产复制、tag 推送、GitHub Release 创建/更新和资产上传。
- 语义查询新增后端结果缓存，同一查询词和同一索引版本会直接返回缓存结果；撤回每次查询都反序列化未常驻超大分片的策略，避免重复查询同一个词仍被数 GB 分片 I/O 拖慢。
- 语义查询热路径增加书籍语义画像候选筛选：对未被常驻加速分片覆盖的图书，先按每本书的 512 维 profile 选出候选，再做段落级暴力搜索，避免不同查询词仍扫描全部未覆盖图书。
- 语义查询 profile 增加内存缓存，并将未覆盖图书的段落级候选数量收紧到 60 本，减少不同查询词反复读取 profile 和扫描过多段落的耗时。

## v1.8.9

v1.8.9 是阅读连续性与公开发布整理版，重点改善图文分页、滚动/整页切换、脚注交互和设置页返回路径，并移除公开仓库中的固定部署信息。

### ✨ 新增 / 改进

- 图书信息中的阅读时间线按日汇总阅读时长、阅读字数和当日最后进度，柱状图按各自峰值比例显示，悬停可查看具体数据。
- 书架横向列表加大图标与行高；封面单击、双击与布局/搜索浮层收起的交互进一步统一。
- 阅读页图片分页支持：当前页优先展示可容纳的图片顶部，下一页从图片顶部完整呈现；图片无需跨页时不会重复显示。
- 语义索引和外置词典设置页关闭后返回常用设置页，避免误退回书架。

### 🐛 修复 / 调整

- 整页与滚动模式切换改为按正文锚点恢复位置，降低切换时前跳、轻微上浮和章节内错位。
- 正文脚注「注」再次点击会收起已展开的注文。
- 阅读进度明细改为“每天最后读到哪里”的日摘要，避免逐章记录长期膨胀。
- 更新检查只使用 GitHub Release；同步服务地址不再内置，需由用户在设置中填写 HTTPS 地址。
- 移除公开仓库中的真实服务器地址、部署路径、系统服务与证书配置，避免发布部署信息。

## v1.8.8

v1.8.8 是 v1.8.7 之后的书架、词典、语义索引和阅读页交互修复版，重点把语义索引迁入主设置，增加外置词典管理与优先查询，并集中修复滚动/双页模式切换、语义索引状态、批注/高亮展示、脚注标记和书架设置页的一批体验问题。

### ✨ 新增 / 改进

- 主设置新增「语义索引」管理页，支持语义模型下载/删除、语义索引和加速索引建立/删除、后台运行、续建、状态本地持久化和占用空间显示。
- 主设置新增「词典」管理页，支持添加词典、删除词典、启用/禁用和调整优先级；外置词典优先于内置词典查询，查词结果显示来源并支持多结果折叠。
- 外置词典导入支持 TSV / CSV / JSON、StarDict `.ifo/.idx/.dict/.dict.dz`、常见未加密 zlib/未压缩 MDX；补完整 MDX/MDD 块读取框架，LZO 或加密 MDX/MDD 会给出明确转换提示。
- 书架筛选面板新增「列数」设置；默认保持封面自适应，手动调整后按固定列数等比缩放封面。
- 阅读页设置新增「仿真翻页」动画选项；双页和滚动模式改成同一行互斥开关，减少模式状态误解。

### 🐛 已修复问题

- 修复语义索引窗口关闭后状态丢失的问题；语义模型、语义索引和加速索引状态会持久化在本地，重新打开设置页可直接显示上次状态。
- 修复进入语义索引页时 UI 卡住的问题；模型、索引和加速索引状态改为快照读取和后台刷新，避免打开弹窗时同步阻塞。
- 修复语义模型和语义索引已就绪时，点击建立加速索引仍重复读取/加载已就绪项的问题；建立加速索引只准备必要的加速索引任务。
- 修复语义索引与加速索引进度不可续建、窗口关闭后任务中断感强的问题；正在运行的任务可继续后台执行并恢复进度显示。
- 修复主设置页语义索引入口、封面显示设置和自动导入目录说明混排的问题；删除多余说明文案，保留原有设置项。
- 修复阅读页滚动模式与整页模式切换时正文轻微位移或跳页的问题；模式切换时按可见锚点恢复当前位置。
- 修复滚动模式自由滚动后再点击翻页，会把上一页末行推进到下一页第五六行的问题；滚动翻页改用真实正文 DOM 与底部裁剪，减少重复行。
- 修复滚动/整页模式下正文脚注「注」大小、颜色不一致的问题；脚注标记统一使用整页模式的视觉规则。
- 修复高亮颜色遮挡文字、让正文发糊的问题；高亮改为更浅的下半部刷痕效果，降低透明色覆盖正文的影响。
- 修复批注/高亮列表展示上下文过长的问题；列表只显示真正高亮的文字，跳转和编辑能力保持不变。
- 修复阅读页设置里「页面」「切换」按钮语义绕的问题；删除旧按钮，改为「双页」「滚动模式」互斥开关。
- 修复词典增强设置浮层受词典释义高度影响、短释义时设置内容被截断的问题；设置面板改为独立浮层，并只保留近义、反义两项常用开关。

### 🔧 工程 / 发布

- 新增 HowNet 词义增强数据源、来源说明和导出脚本，内置词典可补充近义/反义信息。
- 书架全文检索页旧的语义索引建立入口移除，统一迁入主设置页，避免同一任务入口分散。
- 版本号统一升级到 v1.8.8，作为 v1.8 系列词典、语义索引和阅读页交互增强节点。

## v1.8.2

v1.8.2 是 v1.8.1 之后的稳定修复版，重点处理大批量导入、启动响应、自动导入目录和发布检查流程。

### ✨ 新增 / 改进

- 主菜单「导入书籍」支持大批量导入进度提示，导入大量图书时会显示当前进度、已新增数量和当前文件名。
- 拖拽导入与菜单导入统一走同一套批量导入流程，完成后刷新书架并后台更新全文索引。
- 启动性能日志补充到前后端，记录书架加载、封面渲染、账号设置、自动导入、全文索引、更新检查等耗时，方便后续定位卡顿。
- 固定检查脚本继续增强，覆盖 UTF-8、版本一致性、图标资源、CSS 常见错误和 release 文件存在性。

### 🔧 优化 / 调整

- 手动导入的 EPUB / PDF / TXT 等解析、封面提取、指纹计算移到后台阻塞线程执行，并降低后台线程优先级，减少对前台 UI 的抢占。
- 批量导入不再长时间持有书架全局锁，只在去重、插入和保存时短暂加锁。

## v1.5.0

从 v1.0.0 的「能看 EPUB 的本地阅读器」迭代为一个**带语义检索、PDF.js 全功能阅读、高亮批注、朗读、详细统计**的完整阅读器。下面按「新增 / 优化 / 修复」分类汇总这一阶段的全部改动。

### ✨ 新增功能

**检索**
- **书架全文检索**：对整个书架（或选中的若干本）做正文关键词检索，结果在独立窗口展示。底层用 `memchr::memmem` 字节级匹配 + 多线程并行 + 逐书纯文本内存缓存，全库秒搜。
- **语义检索**：基于 `fastembed`（ONNX Runtime）+ `bge-small-zh-v1.5` 向量模型，找「意思相近」而非「字面相同」的内容。

  - **分片索引**：按整本书切片建图，一次只建一片→建图内存恒定，任何机器/任何库大小都不会爆内存（取代了原先会失败的「单图 + 上限」方案）；
  - **按可用内存自动调配**：能装多少分片走近邻就装多少，装不下的书自动退回并行暴力，结果合并；

- 在线朗读 403 Forbidden → 更新 `Sec-MS-GEC-Version` 至 Edge 143 + 补 `ConnectionId`。
- PDF 书签错显示为「章」→ 显示为「页」。
- 天下格局正文字号极小（书内联 font-size）→ 让其继承阅读器字号。
- 末页正文整体右移（Chromium 多栏尾页 padding）→ 占位空栏 + 页数修正。
