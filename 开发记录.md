# 开发记录 / CHANGELOG

鲲鹏阅读器（Kunpeng Reader）开发期间对功能的增删与调整记录。

---

## v1.6.0

在 v1.5.0 的完整阅读器基础上，加入**图书评分体系、多目录自动导入、书架显示设置、新版更新提示**，并对书架封面加载做了显著提速。

### ✨ 新增功能

**评分**
- 在「书籍信息」弹窗给书打分，**支持半星（0.5 刻度）**：点星的左半=半星、右半=整星，再点同一处清除。
- **评分过滤**：书架漏斗面板里按评分筛选（N 星及以上，支持半星）。
- **封面显示评分**：评分小星叠在封面左下角（只读、半星、带描边在深浅封面上都清晰），可在「我的书架」里开关。

**自动导入**
- **多目录自动导入**：在「我的书架 → 自动导入目录」里设置任意多个目录，每次启动自动扫描（含子目录）把其中的电子书加入书架，已在架的不重复添加。

**「我的书架」设置页**（漏斗面板右上角 ⚙）
- 三个封面显示开关：**封面阅读进度 / 封面评分 / 封面下书名**，各自独立开关；网格视图可只显示封面。

**更新提示**
- 启动后台检查 GitHub 最新发行版，有新版时书架顶部弹横幅（查看更新 / 忽略此版本 / 关闭）。
- 「关于」里新增「检查更新」按钮与「本版更新内容」（拉取对应版本的发行说明，带本地缓存以便离线显示）。
- 检查在 **Rust 后端发请求**（避免前端跨域被拦），结构上方便以后扩展更多源。

### 🔧 优化 / 调整

- **书架封面加载大幅提速**（修复"封面比 1.0 还慢"的回归）：
  - 封面 / EPUB 内嵌资源加 `Cache-Control`，再次渲染直接命中缓存、不再走异步协议重取；
  - 封面请求**取到路径后立刻放锁再读盘**（原先读 167KB 图片时一直占着书架全局锁，几百张并发互相阻塞）；
  - 自动导入扫描改为**锁外扫描 + 过滤**，稳态下不再持锁遍历整个目录（原先扫描期间封面全在排队）；
  - 封面 URL 的缓存参数由「stat 文件 mtime」改为「封面版本号」，去掉每次渲染数百次系统调用。
- 网格视图默认**只显示封面**（书名、进度可在「我的书架」开关），更清爽。
- 更新检查由「6 小时节流」改为**每次启动查一次**（避免"刚发新版却不自动弹"）。
- 「关于」版本号、检查更新等统一从后端 `CARGO_PKG_VERSION` 取，单一来源。
- 仓库加 `.gitattributes` 把 PDF.js 标为 vendored，GitHub 语言统计回归真实手写比例。

### 🐞 修复

- 《万历十五年》等书**脚注弹窗空白**：注释 `id` 落在只含角标「[52]」的内联 `<a>` 上、正文是它的兄弟节点；改为取其所在段落的正文。
- 封面首次显示「**黑一下**」：图片字节到达前先露出 `<img>` 黑底；改为命中缓存即直接出图、未缓存时显示占位底色而非黑块。

## v1.5.0

从 v1.0.0 的「能看 EPUB 的本地阅读器」迭代为一个**带语义检索、PDF.js 全功能阅读、高亮批注、朗读、详细统计**的完整阅读器。下面按「新增 / 优化 / 修复」分类汇总这一阶段的全部改动。

### ✨ 新增功能

**检索**
- **书架全文检索**：对整个书架（或选中的若干本）做正文关键词检索，结果在独立窗口展示。底层用 `memchr::memmem` 字节级匹配 + 多线程并行 + 逐书纯文本内存缓存，全库秒搜。
- **语义检索**：基于 `fastembed`（ONNX Runtime）+ `bge-small-zh-v1.5` 向量模型，找「意思相近」而非「字面相同」的内容。
  - 全库 **HNSW 近邻索引**（`instant-distance`）实现毫秒级检索；
  - **分片索引**：按整本书切片建图，一次只建一片→建图内存恒定，任何机器/任何库大小都不会爆内存（取代了原先会失败的「单图 + 上限」方案）；
  - **按可用内存自动调配**：能装多少分片走近邻就装多少，装不下的书自动退回并行暴力，结果合并；
  - 建索引期间**给打开窗口让路**（暂停几秒）+ 后台线程优先级，开书不卡。

**PDF（PDF.js 自渲染）**
- 连续滚动、文字层选择/复制/划词、内置目录、主题（浅/深/护眼）。
- **缩放**：默认按窗口宽度铺满；`－/＋` 细粒度缩放；**记住每本 PDF 的缩放与双页状态**。
- **双页模式**：两页并排，翻页键整对翻。
- **PDF 内搜索**、**PDF 高亮/批注**（按页+归一化坐标）。
- PDF 文字纳入书架检索与语义检索。
- 翻页键精确翻一页（瞬移，避免平滑滚动竞态）。

**高亮 / 批注**
- 选中文字 → 高亮 / 批注 / web 搜索 / 加书签。
- 大批注页：列出全部条目（含上下文，被批注文字高亮），可点编辑、跳转。
- 高亮 ID 改用**内容指纹**（防文件移动导致数据丢失）。

**朗读（TTS）**
- 从当前页开始朗读，**逐词高亮**（CSS Custom Highlight API，不改 DOM），**自动翻页/跳章**。
- 两种音源：**系统语音**（离线，Web Speech API）/ **在线·微软神经**（edge-tts，免费 Azure 级中文音色）。
- 在线音源**流水线预取**下一两句，句间几乎无缝。
- 设置可选音源、音色（晓晓/云希/云扬/晓伊/东北话）、语速。

**阅读统计**
- 由「只存总量」升级为**按（本地日 × 小时 × 书）的小时桶流水**。
- 统计窗口分 **日 / 月 / 年 / 总** 四个视图：时长、字数、读过/读完本数、高亮/批注数；SVG 柱状图（日=24 时段、月=每天、年=每月）；本期读过的书清单。
- 「已读字数」改为**真正读过的字数**（页面停留≥3 秒且逐页前进才计入），不再用「进度 × 总字数」高估。

**其它**
- **更换封面**：选中一本书 → 工具栏「更换封面」→ 选图片替换。
- **关于页**（⋮ 菜单）：软件名、版本、GitHub 链接。
- **沉浸模式**：工具栏淡出但保留占位（切换零重排、绝不跳页），沉浸下右上角阅读统计仍显示。
- **脚注弹窗**：点注释角标就地弹出注释正文，而非跳走。
- **txt 章节化**：网文按「第X章/节/回」自动切章建目录，大文件虚拟化加载（打开秒开）。
- EPUB 与 PDF **各自记忆**阅读窗口的大小/位置。
- 每本书的**页数缓存**：版式一致则复用，免重复测量。
- 选中文字一键 **web 搜索**（系统默认浏览器）。
- 禁用 F12 / 浏览器右键菜单。

### 🔧 优化 / 调整

- **阅读位置稳定性**：顶部锚点改为字符级 caret，并**只在导航时记录、之后冻结**——改字号/边距/缩放时顶部文字不再累积漂移。
- **正文默认两端对齐**：修复部分 EPUB 把对齐写在 `<body>` 上、被阅读器丢弃导致排版异常的问题。
- **逻辑章节按所在文件去重**：金庸全集（每回独立文件）保留到「回」级；Python Cookbook（上千个 #锚点小节）合并回章级，不再显示一千多章。
- **滚动条精确定位**：按全书页位置映射，同章内直接翻页（平滑、不重载），跨章才加载。
- 书架封面整排居中（原先空隙堆在右侧）。
- 「全选」只选当前过滤/搜索后显示的书。
- 工具栏高度 35px、图标居中；书内搜索框改为贴着搜索键的下拉面板。
- 页边距夹到非负且有上限（负边距会破坏分栏排版）。
- 语义索引建立时 CPU 让路 + 后台优先级，打开阅读器不再变慢。
- PDF 默认按窗口宽度铺满，开箱即舒适尺寸。
- 编译：开发期用「快速档」加速迭代，正式发布改回满优化（`opt-level=3, lto=true`）。

### 🐞 修复

- 金庸全集只显示 12 章 → 显示完整目录（数百回）。
- 沉浸模式切换跳页（尤其金庸）→ 保留工具栏占位、零重排。
- 设置面板：每调一个数值就自动关闭；拖滑块时面板消失 → 已修。
- PDF 选中点书签**闪退** + 高亮后叠菜单 → 操作后清空选区。
- PDF PageDown 要按两下、双页模式要按两次 → 瞬移翻页 + 双页整对翻。
- 大 txt（百万字）加载慢 / 加载圈不消失 → 章节化虚拟加载、走合并阅读页。
- 拖动滚动条经过正文时卡住 → 拖动期间 iframe `pointer-events:none`。
- 在线朗读 403 Forbidden → 更新 `Sec-MS-GEC-Version` 至 Edge 143 + 补 `ConnectionId`。
- PDF 书签错显示为「章」→ 显示为「页」。
- 天下格局正文字号极小（书内联 font-size）→ 让其继承阅读器字号。
- 末页正文整体右移（Chromium 多栏尾页 padding）→ 占位空栏 + 页数修正。
